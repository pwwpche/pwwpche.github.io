#####BadUseOfReturnValue :
"()Ljava/lang/String;" ==> String f();<br>
"(ILjava/lang/Class;)J" ==> long f(int i, Class c);<br>
"([B)V" ==> void String(byte[] bytes);<br>
Reference: JNI Type Signatures(https://docs.oracle.com/javase/1.5.0/docs/guide/jni/spec/types.html)
